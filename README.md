\# UIDAI Data Hackathon – Aadhaar Analytics



\## Objective

Analyze Aadhaar enrolment, biometric, and demographic datasets to extract actionable insights and support administrative decision-making.



\## Datasets

\- Enrolment data

\- Biometric activity data

\- Demographic update data



(All data is aggregated and privacy-safe.)



\## Project Structure

\- notebooks/ → Analysis pipeline

\- data/ → Cleaned datasets

\- ppt/ → Final presentation



\## Workflow

1\. Data loading

2\. Data cleaning

3\. Univariate analysis

4\. Bivariate analysis

5\. Trivariate analysis

6\. Anomaly detection

7\. Insights \& recommendations



\## How to Run

Open notebooks sequentially in the `notebooks/` folder.
